# 🔐 Password Manager (Tkinter + Python)

## 📌 Project Description
This is a GUI-based Password Manager built using Python Tkinter.  
It allows users to store, search, delete, and copy passwords securely using encryption.

---

## 🚀 Features
- Login system (Master password)
- Add new password entries
- View saved passwords
- Delete entries
- Search functionality
- Copy password to clipboard
- Encryption using Fernet
- SQLite database storage

---

## 🛠 Tools Used
- Python
- Tkinter (GUI)
- SQLite (Database)
- Cryptography (Encryption)
- Git & GitHub
